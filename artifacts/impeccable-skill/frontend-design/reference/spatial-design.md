# Spatial Design

- Create rhythm with alternating tight and generous spacing.
- Let important regions breathe; density should be earned by relevance.
- Use asymmetry, offsets, and unequal columns to create direction and memory.
- Flatten container usage. Separate sections with rules, background shifts, or spacing before reaching for more boxes.
- When cards are needed, vary their role and scale instead of repeating one template everywhere.
- Align text blocks to a strong reading edge; left alignment usually feels more intentional for tools.
- Break the grid deliberately only when it strengthens hierarchy.
