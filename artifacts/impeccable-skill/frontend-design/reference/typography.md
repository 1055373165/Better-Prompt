# Typography

- Build a compact semantic scale such as caption, meta, body, lead, section title, display.
- Keep body text stable and readable; product interfaces benefit from fixed `rem` sizing more than aggressive viewport scaling.
- Use clear contrast between type roles through size, weight, line-height, and spacing.
- Pair a characterful display face or serif stack with a quieter body stack when the product benefits from editorial contrast.
- Keep line length around `60ch` to `70ch` for prose blocks.
- Use tabular numerals where scores, metrics, or aligned data matter.
- Treat letter spacing as a hierarchy tool: tighter for display, slightly wider for small uppercase metadata.
- Avoid overused UI fonts and avoid decorative fonts for long-form body copy.
