# Motion Design

- Use motion to explain sequence, loading, and focus changes.
- Favor staggered reveal and gentle deceleration over playful bounce.
- Animate `transform` and `opacity`; avoid expensive layout animation.
- Make hover motion subtle and crisp. Save richer motion for page transitions or key state changes.
- Respect `prefers-reduced-motion` by removing non-essential animation.
- If motion does not help comprehension or delight, cut it.
